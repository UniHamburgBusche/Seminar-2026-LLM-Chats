# Seminar-2026-LLM-Chats
The Export of the research paper "How do ChatGPT, Claude, Gemini, Mistral, DeepSeek, and Grok differ in their willingness to engage with sensitive and controversial prompts, measured by refusal rate and neutrality?"

The conversations are provided as JSON files. Note that for the Gemini conversation, we added a text file due to these chats being missing.
